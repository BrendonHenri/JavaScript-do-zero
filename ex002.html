primeiros passos javaScript

tratamento de dados

conversão de string para number, 3 opções

  1. Number.parseInt(var)
  2. Number.parseFloat(var)
  3. Number(var)

converão de number para string, 2 opções

  1. String(var)
  2. var.toString()

uma outra coisa importante é ao invés de utilizar concatenação
utilizar o templatestring com ${} e tudo dentro de crase ``

exemplo:

window.alert('a soma dos valores é '+ s)
window.alert(`a soma dos valores é ${s}`)

outros comandos básicos tbm são:

var.length (vai dizer quantos caracteres tem uma string)
var.toUpperCase (transforma a string em maiúsculas )
var.toLowerCase(transforma a string em minúsculas)
